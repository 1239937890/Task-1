学习笔记 :

用xhtml展示文档结构，用css进行可视化表达

一、属性与标记之间、各属性之间需要以空格隔开。属性值以双引号括起来。

二、编写XHTML的规范（重构 才是 XHTML 真正的意义）：

  1. 所有标记元素都要正确的嵌套，不能交叉嵌套。
  2. 所有的标记都必须小写。
  3. 所有的标记都必须关闭。

三、在每条声明的末尾都加上分号，这么做的好处是，当你从现有的规则中增减声明时，会尽可能地减少出错的可能性

四、使用CSS来决定元素的外观。

不要在属性值与单位之间留有空格，因为在某些浏览器中无法正常工作
